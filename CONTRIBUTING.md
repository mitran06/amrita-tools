# Contributing to Amrita Tools

Hey there,
Thanks for checking out **Amrita Tools**, i'm glad you're interested in contributing.

## What You Can Contribute

You’re welcome to add **anything** that helps make building Amrita-related projects easier. Some examples include:

* Reusable **scripts** and command line utilities
* UI components, widgets, or helpers
* Backend modules or services
* Tools written in **any programming language**
* Implementation to existing tools in other languages
* Improvement of existing tools in existing languages
* Code comments, docs, or examples that help others understand things better

If it saves time and could be useful to other devs at Amrita, it’s welcome here.

## How to Contribute

1. **Fork this repo** and create your feature branch 
`git checkout -b my-new-feature`
2. Add your tool/script/module under the appropriate language directory (or create a new one)
3. Add a `usage` file to show how it works.
4. Update the `README.md` with info about your tool.
5. Test your code locally.
6. Submit a **Pull Request**

---

## Good to Knows

* Keep folder names descriptive
* You’ll automatically show up in the contributors list once your PR is merged
* Open an issue if you have any suggestions or ideas
